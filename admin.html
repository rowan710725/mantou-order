📋 如何生成 Token？點這裡展開步驟
1. 打開 github.com 並登入
2. 右上角頭像 → Settings
3. 左側最下方 → Developer settings
4. Personal access tokens → Tokens (classic)
5. 點「Generate new token (classic)」
6. Note 填：方王商品後台
7. Expiration 選「No expiration」
8. 勾選 public_repo（只需這一個）
9. 點最下方「Generate token」
10. 複製那串 ghp_ 開頭的 token（離開頁面就看不到了）
11. 貼到上方欄位，點「儲存」
